# Zeitkompass-Forschung
Dokumentensammlung zu den Forschungsarbeiten zum Assistenzsystem Zeitkompass.

Diese Dokumentensammlung umfasst zentrale Forschungsdokumente, Evaluationsberichte, Studienprotokolle sowie ergänzende Materialien zur Entwicklung, Validierung und Anwendung des Assistenzsystems Zeitkompass. Sie dient der strukturierten Darstellung der wissenschaftlichen Grundlagen, empirischen Befunde und nutzerzentrierten Entwicklungsprozesse, die das System über mehrere Projektphasen hinweg begleitet haben.

Stand: 17.11.2025
